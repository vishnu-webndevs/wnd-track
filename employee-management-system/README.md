### Key Responsibilities of this Directory:
* **`app/Http/Controllers/`**: Secure controllers for the desktop tracker API endpoints and live streaming signaling.
* **`app/Services/TelegramService.php`**: Telegram bot integration for automated tracking log notifications.
* **`database/`**: Migrations and schemas for timesheets and user administration.
